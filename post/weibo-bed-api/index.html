新浪图床API修复

Posted on 2020-03-13，1 min read

目前网络上的现存的新浪图床API几乎都无法使用。主要原因还是新浪已经废弃了那个网址。
现在可以使用的图片提交由以前的：

http://picupload.service.weibo.com/interface/pic_upload.php

改为了：

https://picupload.weibo.com/interface/pic_upload.php

如果你手上恰巧有一份新浪图床API的源码的话，只需要将其中的上传网址替换成下面这个就可以了。
相关的上传源码网上有很多现存，此处就不提供了。
顺便一提，目前tva[1-4]以及tvax[1-4]还不会被403，至于还能稳定多久就不确定了。
