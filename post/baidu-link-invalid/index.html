Posted on 2019-08-17，2 min read

前言

闲得无聊逛Gayhub的时候偶然看到一个项目，貌似是用来防止链接被和谐的。然后就下载来试了试。发现还真的挺有用的。

项目地址：https://github.com/logonod/Panlian

大多数人都应该知道百度有一套秒传文件的机制吧。一个相同文件的MD5值是相同的，因此在上传的过程中通过MD5可以对文件进行匹配，达到秒传的目的。而这个项目目测好像就是利用的这个来实现秒传的用途的。
