    前言

    如果你是一个维修电脑的，或者经常去帮别人装电脑或者修复系统的话。那么你会时常有个烦恼，机主的新系统什么都没有，导致需要花上很多时间去装一些不必要或者说用不了多久就会卸载的软件。像我，就经常被这些问题所困扰。但是单文件程序的出现，很大程度上的解决了这一个问题。

    所谓的单文件，是指那些从一个单独的应用程序之中抽离出来，或者将程序压缩成一个可执行文件(exe)的应用程序。这类文件普遍的特点是：

    1.文件大小小

    2.只有一个文件

    3.运行是将程序主题释放在内存或者硬盘上，待关闭后自动回收

    从上面可以看出，单文件的程序有着快捷，方便的特点。不用想普遍的应用程序一样会经历一个卸载的过程。所以说单文件程序也就成为了我特别喜欢的一类。装在U盘上面简直爽的不要不要的。

    本文着重更新那些网络上收集或者说自己经常在使用的单文件程序。由于出处较多，也不好注明来源，故以下所有资源来源于网络。如有侵权请文章留言，我将会删除。

    在未声明情况下，文章内所有程序均为64位应用程序，请自行确认。
