  • йҖҡеёёжғ…еҶөдёӢе…ій—ӯжӣҙж–°дјҡеҸ«дҪ еңЁservices.mscдёӯжүҫеҲ°Windows UpdateиҝҷдёӘжңҚеҠЎпјҢз„¶еҗҺе°Ҷе…¶ж”№жҲҗзҰҒз”ЁгҖӮдҪҶжҳҜдәӢе®һдёҠпјҢwin10дёәдәҶйҳІжӯўз”ЁжҲ·зҰҒз”ЁиҮӘеҠЁжӣҙж–°иҝҳжҗһдәҶдёҖдәӣйӘҡж“ҚдҪңпјҢжүҖд»ҘеҜјиҮҙиҝҷдёӘе№¶дёҚиғҪе®Ңе…ЁзҰҒз”Ёwin10зҡ„иҮӘеҠЁжӣҙж–°гҖӮе…·дҪ“зҡ„иҜқжҲ‘д»¬иҝҳйңҖиҰҒжҠҠеҸҰеӨ–дёҖдёӘжңҚеҠЎWindows Update Medic Serviceд№ҹзҰҒз”ЁжҺүпјҢдҪҶжҳҜиҝҷдёӘжңҚеҠЎзҰҒз”Ёзҡ„ж—¶еҖҷдјҡжҳҫзӨәжӢ’з»қгҖӮпјҲжӯӨж•ҷзЁӢеҜ№еә”win10дё“дёҡзүҲпјҢе…¶д»–зүҲжң¬еҸҜиғҪдјҡжңүдәӣеҮәе…Ҙпјү
  • йҰ–е…ҲпјҢwin + rиҝӣе…ҘиҝҗиЎҢпјҢиҫ“е…Ҙregeditиҝӣе…ҘжіЁеҶҢиЎЁзј–иҫ‘еҷЁгҖӮ
  • е®ҡдҪҚеҲ°и·Ҝеҫ„пјҡи®Ўз®—жңә\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\WaaSMedicSvc

  • иҝҷж—¶еҖҷеҶҚжү“ејҖservices.mscе°ұдјҡеҸ‘зҺ°Windows Update Medic Serviceд№ҹиў«зҰҒз”ЁжҺүдәҶгҖӮжӯӨж—¶еҶҚе°ҶWindows Updateд№ҹзҰҒз”ЁжҺүпјҢиҝҷж ·е°ұе®Ңе…Ёе…ій—ӯwin10иҮӘеҠЁжӣҙж–°дәҶгҖӮ
