Win10各种配置

Posted on 2021-05-29，2 min read
  • 记录一些win10的配置，方便以后查询。

关闭自动更新

  • 通常情况下关闭更新会叫你在services.msc中找到Windows Update这个服务，然后将其改成禁用。但是事实上，win10为了防止用户禁用自动更新还搞了一些骚操作，所以导致这个并不能完全禁用win10的自动更新。具体的话我们还需要把另外一个服务Windows Update Medic Service也禁用掉，但是这个服务禁用的时候会显示拒绝。（此教程对应win10专业版，其他版本可能会有些出入）
  • 首先，win + r进入运行，输入regedit进入注册表编辑器。
  • 定位到路径：计算机\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\WaaSMedicSvc
  • 将Start项修改为16进制的4（注，也可以win + x点开Windows powershell 管理员，然后输入命令REG add HKLM\SYSTEM\CurrentControlSet\Services\WaaSMedicSvc /v Start /t REG_DWORD /d 4 /f可以直接修改注册表）
  • 这时候再打开services.msc就会发现Windows Update Medic Service也被禁用掉了。此时再将Windows Update也禁用掉，这样就完全关闭win10自动更新了。
  • 右键此电脑 -> 管理 -> 系统工具 -> 任务计划程序 -> 任务计划程序库 -> Microsoft -> Windows -> WindowsUpdate -> 禁用里面的触发器
  • 然后删除Win10下载的更新文件
    • C:\Windows10Upgrade
    • C:\Windows\UpdateAssistant
    • C:\Windows\UpdateAssistantv2
    • C:\Windows\SoftwareDistribution\Download
    • C:\Windows\SoftwareDistribution\DataStore

关闭权限提示

  • 对于带有如下图标的，进行操作的时候可能会弹出一个弹窗确认。
  • 禁用方法：
    • 右键 -> 此电脑 -> 安全和维护 -> 更改用户账户控制设置 -> 拉到最低，然后点击确定。

关闭快速访问

  • 快速访问会记录最近访问的文件历史，但是这属于隐私，所以需要禁用。
